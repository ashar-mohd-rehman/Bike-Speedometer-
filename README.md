# Bike-Speedometer-
A digital speedometer for my mtb based on XIAO esp 32 s3, NEO 6M GPS module and an 1.3 inch OLED display

Let us talk about features 
<img width="976" height="662" alt="Screenshot 2026-04-10 203553" src="https://github.com/user-attachments/assets/3d6eace0-13ae-43df-bee1-d5914aad0166" />

- It shows the speed of the cycle (mtb)
- Distance covered from the starting point
- Trip timer which shows how long u have been riding for
- Altitude, which shows ur alt from sea level
- Today's Date
- Satellites GPS module is connected to
- And last but not the least. The Battery Percentage
